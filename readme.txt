A e-mail contact form block plugin

== Description ==

A e-mail contact form block plugin. Add the block to your page or post. Form collects name, email, phone, prefferred contact method and a short message. When submitted on front end, sends a email to address set up in admin options page. Output shown to user after submit is also set up via admin options page. Fonts, Colors, and spacing can be customized using page editor side panel. 

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/cpj-contact-form` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. In page or post editor click add new block, search for 'cpj' and select cpj-contact-form.

== Frequently Asked Questions ==

= I am not getting email when someone submits form =

Check you have a valid email address entered in e-mail field in admin options
